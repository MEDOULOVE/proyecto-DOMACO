# proyecto-DOMACO (DM)

Proyecto desarrollado por DoMaCo Group (Dolcin Management Company), orientado al rubro de ferreterías bajo el nombre comercial DOMACO (Dolcin Matériaux de Construcción).

Inicialmente, el proyecto fue desarrollado bajo el nombre FERREMAS, el cual contemplaba una estructura más amplia,
con una mayor cantidad de microservicios y funcionalidades más completas en el frontend. 
Sin embargo, durante el desarrollo, el jefe del proyecto decidió cambiar el enfoque y establecer una nueva identidad para la ferretería, adoptando el nombre DOMACO.

Debido a este cambio, se realizaron modificaciones y adaptaciones sobre la estructura inicial del sistema con el objetivo de ajustarlo a los nuevos requerimientos y necesidades definidas para DOMACO.

Posteriormente, se implementaron mejoras y ajustes específicos para cumplir con los requerimientos funcionales solicitados para el nuevo contexto del proyecto.

La documentación completa del proyecto se encuentra disponible dentro de la carpeta principal del sistema.
