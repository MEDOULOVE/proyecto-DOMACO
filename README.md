# proyecto-DOMACO (DM)

Proyecto desarrollado por DoMaCo Group (Dolcin Management Company), basado en dos ferreterías: DOMACO (Dolcin Matériaux de Construcción) y FERREMAS.

El sistema de FERREMAS contaba con más microservicios y funcionalidades completas, el cual fue utilizado como base para el desarrollo.

Posteriormente se realizaron modificaciones y adaptaciones para cumplir específicamente con los requerimientos solicitados por DOMACO.

La documentación completa del proyecto se encuentra dentro de la carpeta principal del sistema.
